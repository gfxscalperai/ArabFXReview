دليل الإعداد والنشر

① Supabase — قاعدة البيانات
1. افتح supabase.com وأنشئ مشروعاً مجانياً
2. اذهب إلى SQL Editor وشغّل ملف supabase-schema.sql
3. احفظ الـ URL و Service Key من الإعدادات
② GitHub — رفع الكود
1. أنشئ مستودعاً جديداً على GitHub
2. ارفع جميع ملفات المشروع
3. تأكد أن index.html في المجلد الجذر
③ Vercel — النشر والاستضافة
1. افتح vercel.com واربط حساب GitHub
2. استورد المستودع
3. أضف المتغيرات البيئية التالية:
CLAUDE_API_KEY = sk-ant-...
SUPABASE_URL = https://xxx.supabase.co
SUPABASE_SERVICE_KEY = eyJ...
CRON_SECRET = اخترأي_كلمة_سر_عشوائية
✓ بعد النشر
سيعمل الموقع على yourname.vercel.app
الأخبار ستُجلب تلقائياً كل ساعة
